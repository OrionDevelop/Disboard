Misskey API wrapper for .NET Standard 2.0.  
Based on Misskey 8.45.1.


## Caution

**This is a generic Misskey API wrapper**, We do not implement features unique to the instance.  


## Usage

```csharp
var misskey = new MisskeyClient("misskey.xyz");
```
