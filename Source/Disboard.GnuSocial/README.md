GnuSocial API wrapper for .NET Standard 2.0.  
Based on GnuSocial 1.2.x.


## Note

**This is a generic GnuSocial API wrapper**, we do not implement features unique to the instance.  


## Usage


```csharp
var gs = new GnuSocialClient("freezepeach.xyz");

```
